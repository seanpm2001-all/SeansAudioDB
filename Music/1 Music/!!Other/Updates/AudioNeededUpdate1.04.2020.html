About this file - documentation

As a wishlist

This is a wishlist for new music and sound effects that I plan to purchase/download to my computer and add to my music collection. It serves as a wishlist, archiving the links to the songs I want to get.

Indexing

Songs and sound effects are counted on a daily basis whenever music or sound effects are added to the list.

Music is counted without a decimal, for example, a single song at the beginning of a list would have a (1) at the end of its name.

If a playlist came after that which contained 40 songs, there would be a bubble counting up the items in the playlist, and the previous song count, meaning a (41) would be at the end

Sound effects are a different category. They start out at (1.1) if there is more than 1 sound effect in a day, it will follow by (1.2) if a collection of sound effects were added, and the collection had 93 sound effects, it would be followed by (1.95) and so on

Counters

There are several counters across this file. On each daily list, where I add new content, there is a sound effect counter, and a song counter, counting how many songs and sound effects I added to the list. There are counters for days I added in to the list, how many changes I made, and total song and sound effect counts

TXT document style

This text document is styled uniquely. It has some python syntax in it.

As of the November 17th 2019 build, it is being converted to HTML5.

New entries

A new entry starts with the name of the song or sound effect, followed by its daily index number. Under that, is the link to the audio. If it is in a playlist, a counter of how many items in the playlist will be under the link, if not, a file path will be shown.

The file path starts with 1 of 2 different syntax:

(Add to: Music/ - this is for adding in music

(Add to: SFX/ - this is for adding in sound effects

Afterwards, the desired file path will be shown after that, not including the file names. Older versions didn't have the Music/ and SFX/ headers, they just had the (Add to: example/example)
